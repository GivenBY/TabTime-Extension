## Tab Time Tracker

A Chrome extension to track time spent on each website.

### Installation (Chromium-based browsers)

1. Open your browser.
2. Go to `{browser}://extensions/`.
3. Enable **Developer mode** (top-right toggle).
4. Click **Load unpacked**.
5. Select the extension's root folder.
6. The extension loads immediately and pin it to toolbar.
